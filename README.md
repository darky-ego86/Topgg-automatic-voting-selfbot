# Top.gg Auto Voter - by darky.ego

Cleaned & Improved Version with Proxy + Multi Bot Support

### Features
- Support for multiple bots
- Proxy support (HTTP & SOCKS5)
- Better error handling
- Clean and modern code
- Headless mode option

### Setup Guide:

1. **Install Dependencies**
   - Double-click `setup.bat`

2. **Add Your Tokens**
   - Open `output/verified.txt`
   - Add your Discord tokens (one per line)
   - How to get token:
     - Open Discord in browser
     - Press `Ctrl + Shift + I`
     - Go to Console tab
     - Type: `console.log(localStorage.token)`
     - Copy the token (without quotes) and paste it

3. **Configure (Optional)**
   - Open `index.js`
   - Edit the `CONFIG` section to add more bots or proxies

4. **Run the Script**
   - Double-click `start.bat`

### Important Warnings:
- Use only alt accounts (Never use your main account)
- This script violates Discord & top.gg Terms of Service
- You may get IP banned → Use proxies
